Веб-приложение для управления временем по технике Pomodoro с отслеживанием статистики работы.

# Возможности

### Таймер
- Режимы «Работа» (25 минут) и «Отдых» (5 минут)
- Редактирование времени
- Старт, пауза, сброс

### Темы

- Светлая тема (градиентный оранжево-розовый фон)
- Тёмная тема (тёмно-синий градиент с фиолетовыми акцентами)
- Сохранение выбранной темы в localStorage

### Звуковые настройки

- Отслеживание завершённых рабочих сессий
- Общее время работы за день
- Средняя продолжительность сессии
- Статистика за день / неделю / месяц в календаре

### Календарь и выбор периода

- Просмотр статистики за выбранную дату, неделю или месяц

### Локализация

- Поддержка русского и английского языков
- Переключение по кнопке в сайдбаре
- Все тексты, дни недели и месяцы адаптируются под выбранный язык
