# Фичи проекта:

1) Таймер с отсчётом
2) Настраиваемый сигнал таймера
3) Сохраняемая статистика таймеров за сегодня, неделю, месяц итд
4) Иконка приложения


# Внешний вид

Шапка: Лого + название, быстрая статистика

Слева: кнопки управления:
   - переключить тему
   - Включить, отключить плеер

Основная часть: отсчёт времени, снизу кнопка старта и серым кнопка остановить(меняются при нажатии), справа от них кнопка для настройки плеера(включает модалку с настройками)

Ниже: Расширенная статистика: 

Счётчик помодоро за сегодня
    - Номер текущей сессии
    - Общее время работы сегодня
    - Прогресс к дневной цели

Выбором конкретной даты

# Как сделать фичи:


## кнопки над таймером:

1) При нажатии на Отдых, меняется цвет на чёрный - через добавление active.
2) Время на таймере меняется на 05:00 - меняем текстконтент span

## таймер

1)  при нажатии на него время меняется через скрол
    через кнопки вниз и вверх
    появляется каретка в месте клика

2) Если выбираешь время больше 59 минут, то цвет часов меняется на белый и прибавляется 1
